# clankercloud/homebrew-tap

Homebrew tap for Clanker Cloud tooling.

Upstream Clanker CLI repo: https://github.com/bgdnvk/clanker

## Install

```bash
brew tap clankercloud/tap
brew install clanker
```

## Add more formulae

Put additional formula files in `Formula/`, e.g. `Formula/another_tool.rb`.
